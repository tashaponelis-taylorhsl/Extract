Thai Ledger — Privacy Policy

Last updated: March 17, 2026

1. Overview

Thai Ledger ("the Application") is a private internal accounting tool. This Privacy Policy explains how the Application handles data when used by authorized personnel to process accounting documents and post transactions to QuickBooks Online.

2. Who We Are

Thai Ledger is a private internal application used exclusively by authorized accounting staff. It is not a public application and is not available to the general public.

4. Third Party Services

The Application uses the following third party services:

  • Anthropic Claude API — Document text is sent to Anthropic's API for AI processing and data extraction. Anthropic's privacy policy applies: anthropic.com/privacy
  • Intuit QuickBooks Online API — Transaction data is posted to QuickBooks Online via Intuit's API. Intuit's privacy policy applies: intuit.com/privacy
  • Microsoft Graph API — Document folders in Teams/SharePoint are accessed via Microsoft's API. Microsoft's privacy policy applies: privacy.microsoft.com

5. Data We Do Not Collect

The Application does not: (a) store financial documents on any server; (b) retain accounting data beyond what is posted to QuickBooks Online; (c) share data with any party other than the services listed above; (d) use data for advertising or marketing purposes; (e) sell or transfer data to third parties.

6. Data Security

All API communications use HTTPS encryption. OAuth tokens are stored only in the user's local browser and are never transmitted to our servers. The Vercel proxy server handles API routing only and does not log or store financial data.

7. Data Retention

The Application does not retain data. All processing occurs in the user's browser session. OAuth tokens stored in localStorage remain until the user clears their browser data or explicitly disconnects.

8. User Rights

Users can delete all locally stored data at any time by clearing their browser's localStorage or using the Application's built-in disconnect features.

9. Changes to This Policy

This policy may be updated periodically. Continued use of the Application constitutes acceptance of any changes.

10. Contact

For privacy-related questions, please contact the application administrator.
